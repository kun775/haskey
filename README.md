# haskey

Linux 系统常用命令合集，通过交互式菜单快速安装、配置和管理系统常用工具。

## 使用方法

```bash
bash haskey.sh
# 或
chmod +x haskey.sh && ./haskey.sh
```

## 功能列表

- **zsh 管理**
  - 一键安装 zsh + Oh My Zsh，并设为默认 shell
  - 配置 zsh 主题（ys、robbyrussell、agnoster 等 14 个常用主题）
- 网络工具（待开发）
- 系统监控（待开发）
- Docker 管理（待开发）
- 开发环境（待开发）

## 支持的包管理器

apt / yum / dnf / pacman / zypper，脚本会自动检测。

## 环境要求

- Linux / WSL / macOS
- bash 4.0+
- curl（安装 Oh My Zsh 时需要）
